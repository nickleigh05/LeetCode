# Arithmetic Operators

```python
5 + 3     # 8
5 - 3     # 2
5 * 3     # 15
5 / 3     # 1.666...  (true division, always float)
5 // 3    # 1          (floor division)
5 % 3     # 2          (modulo/remainder)
5 ** 3    # 125        (exponent)
```

`/` always returns a float, even `4 / 2 == 2.0`. Use `//` when you need an int result (array indexing, midpoints).
