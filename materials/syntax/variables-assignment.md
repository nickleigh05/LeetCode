# Variables & Assignment

```python
x = 5
name = "leetcode"
x, y = 1, 2        # multiple assignment, one line
x = y = 0           # chained assignment, both bound to 0
```

A variable is just a name bound to an object. Python has no type declaration — the name can be rebound to any type at any time.

```python
x = 5
x = "now a string"   # totally legal, x just points somewhere new
```
