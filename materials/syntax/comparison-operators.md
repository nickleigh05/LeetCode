# Comparison Operators

```python
5 == 5     # True  — equality
5 != 3     # True  — inequality
5 > 3      # True
5 >= 5     # True
5 < 3      # False
5 <= 5     # True
```

Returns a `bool`. Works on any orderable type — numbers, strings (lexicographic), tuples (element-by-element). `==` checks value equality, not identity — see [identity-operators](identity-operators.md) for the `is` distinction.
