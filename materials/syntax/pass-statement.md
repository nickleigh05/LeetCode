# `pass`

```python
def todo():
    pass          # placeholder — does nothing

if x > 0:
    pass          # explicitly "do nothing" branch
```

Python requires a body after `:` — `pass` is a no-op statement that satisfies that requirement when you don't want to do anything yet (stub functions, empty class bodies, intentionally-empty branches).
