# Arrays & Hashing Practice Drills

This directory contains progressive practice drills to help you master arrays and hashing concepts.

## Drill Structure

Each drill focuses on a specific concept and builds upon previous knowledge:

1. **Drill 01** - Basic Array Traversal & Hash Set Operations
2. **Drill 02** - Hash Map for Frequency Counting
3. **Drill 03** - Two-Pointer Techniques with Arrays
4. **Drill 04** - Hash Map for Index Mapping (Two Sum Pattern)
5. **Drill 05** - Multiple Hash Sets/Maps
6. **Drill 06** - Array Manipulation with Hashing
7. **Drill 07** - String Hashing Patterns
8. **Drill 08** - Advanced: Sliding Window with Hashing
9. **Drill 09** - Advanced: Prefix Sum with Hashing
10. **Drill 10** - Challenge Problems

## How to Use These Drills

1. Start with Drill 01 and work sequentially
2. Read the problem description carefully
3. Try to solve it yourself before looking at hints
4. Each drill includes:
   - Problem statement
   - Example test cases
   - Time/Space complexity targets
   - Hints section (try without hints first!)
   - Template code to get you started
